# Windows Logging for SOC Investigation

This project demonstrates analysis of Windows event logs and Sysmon telemetry to identify suspicious activity on an endpoint. The investigation focuses on process creation events, authentication logs, and common indicators used in SOC environments.

Skills demonstrated:

• Windows event log analysis  
• Sysmon telemetry investigation  
• Process creation monitoring  
• Authentication log analysis  
• MITRE ATT&CK mapping
